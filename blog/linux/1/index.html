1. What is Linux?

1.1 Definition
It is an open-source monolithic kernel with GNU GPL v2 licence, used as the base of many operating systems.

What is a monolithic kernel? The kernel is software that has hardware control, manages and optimizes system resources such as RAM allocation, CPU processes, drivers, etc. Acting as a bridge between applications and hardware. It is monolithic when all system services operate in the systemâ€™s kernel with syscalls.

What is open-source? It is software where anyone can see, analyze and therefore modify the source code.

What's the GPLv2 license? It is a software license where any modification to the code must have the same license and for every binary file that is distributed, there has to be available source code.
1.2 History
It was created in 1991 by the student of the University of Helsinki Linus Torvalds.

At the same time Richard Stallman along with his GNU project of the Free Software Foundation (FSF) had already created the applications and utilities of an operating system.

These applications and utilities were added to the Linux kernel to create a complete system called GNU/Linux.
