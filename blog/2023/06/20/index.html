Linux advantages

2023/06/20
Si algo he aprendido en mi historia como estudiante y usuario del internet, es que cualquier cosa se puede hacer en linux y de mejor manera, por ejemplo: programar, escuchar musica, automatizacion de actividades, etc. Si tuviera que describir a linux en una palabra seria "libertad", libertad de modificar, de codigo, de personalizar, de precio y en privacidad.

Al usar cualquier sistema operativo con linux, llegaras a usar el manejador de paquetes del sistema, los cuales en su mayoria tienen apliciones con licencias de codigo abierto, por lo que puedes asegurar por ti mismo que ninguna aplicacion llamara a casa sin tu permiso. Tambien linux suele funcinar mejor en un disco duro en comparacion con Windows o macOS y tambien los ambientes escritorio suelen usar menos memoria ram en linux, lo que puedes configurar un sistema operativo linux a tus necesidades de utilidad y de hardware para aprovechar al maximo una computadora ya sea antigua o moderna, en mi caso revivi una laptop hp touchsmart tx2, con un amd turion de dos nucleos y tres poderosisimos gb de ram ddr2, le instale artix linux con runit y dwm como mi window manager, por lo que ahora utilizo esta laptop para tomar notas con xournal++ para la universidad usando menos de 500 mb de ram.
If I've learned anything in my story as a student and internet user, it's that anything can be done in linux and better, for example: program, listen to music, automate activities, etc. If I had to describe linux in a serious word "freedom," freedom to modify, of code, of personalization, price and privacy.

When using any linux operating system, you will come to use the package handler of the system, which in its most part have applications with licenses of open code, So you can assure for yourself that no application will call home without your permission. Also linux usually funncer better on a hard drive compared to Windows or macOS and Desktop environments also usually use less ram memory in linux, which you can configure a linux operating system to your utility and hardware needs for take advantage of the maximum a computer whether old or modern, in my case a laptop hp touchsmart tx2, with an amd turion of two nucleos and three powerful gb gb ddr2, le install artix linux with runit and dwm as my window manager, so now I use this laptop to take notes with xournal++ for university using less than 500 mb ram.
